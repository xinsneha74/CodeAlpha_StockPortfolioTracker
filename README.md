# Stock Portfolio Tracker

## Description
This project is a simple Stock Portfolio Tracker developed in Python. Users can enter stock names and quantities, and the program calculates the total investment value using predefined stock prices. The final result is displayed on the screen and saved to a text file.

## Features
- User input for stock names and quantities
- Predefined stock price dictionary
- Automatic investment value calculation
- Save results to a text file
- Simple command-line interface

## Technologies Used
- Python
- Dictionaries
- File Handling
- Loops
- Conditional Statements

## How to Run
1. Install Python on your system.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:
   ```bash
   python stock_tracker.py
